    knowledge areas

    This section presents the development areas in which I have worked throughout my learning trajectory, these are listed below with a brief description.

    Data Scientist

    I have worked in the analysis of large volumes of data, including user profile data, satellite images, banking sources, among others. In these data, data cleaning and profiling of the same have been developed among other statistical activities.

    Data Analyst

    I worked on the analysis of data to create descriptive analysis, prescriptive analysis and predictive analysis reports. These reports allow us to identify and correlations, outliers and dependencies. Also, I have developed visualizations for effective decision making.

    Data Engineer

    I have worked in the development of applications such as recommendation systems and pages that integrate machine learning pre-processing for the analysis of user profiles, traffic analysis in Colombia, among others. These deployments have been developed on AWS and Heroku.

    SIG Analyst

    I have worked in the analysis of satellite images and remote sensing processing for the classification of land cover, through the Open Data Cube, which is an initiative that allows the processing of large volumes of satellite data for the creation of multi-temporal analyzes.

    Machine Learning

    I have worked on the construction of machine learning models for the classification of different covers of the land surface, among them are Rice, and coffee in the department of Tolima in Colombia, these models have been built and applied on satellite images. Likewise, I have built predictive models for the analysis and behavior of users. Also, I have built models for recommending content based on user preferences and building recommendation systems.

    Python Developer

    I have built Web applications in Python, Django with the integration of HTML5, CSS3 and Javascript for the visualization of user recommendations, transit applications and among others. Likewise, I have worked Sigle Web pages for the dissemination of events and research forums in different areas of computer science.
