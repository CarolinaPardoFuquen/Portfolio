Master's Degree Project detail

The creation of new sources of geographic information generates billions of data periodically and the analysis of large volumes of geographic information for the extraction of main characteristics is an area that is experiencing exponential growth. In this way, one of the most worked domains currently focuses on the extraction, segmentation and classification of elements. These can be objects, vegetation, urban areas, crops, among others. However, this activity involves processing that impacts the use of qualified human and computational resources. The use of these resources will only allow the classification of a small data sample and, given the speed of the increase in information, it is necessary to implement techniques that allow the automation of these processes. From this need, a methodology is created that allows abstracting the main steps for the detection of different crops by means of machine learning techniques. The foregoing generates an advance in remote sensing or remote sensing processes in Colombia. Processing power has always been a challenge in analysis processes. However, through the integration of the Open Data Cube (ODC) as a support tool, the computational requirements necessary for the integration of algorithms and the execution of multitemporal analyzes are linked.
